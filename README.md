# 设计模式：游乐园模拟项目

​	源码在src目录下的main和test文件夹内。

​	test文件夹内为各个设计模式的单项测试，可以通过在该项目的目录下通过终端输入以下命令运行各项测试：

```powershell
mvn test
```
